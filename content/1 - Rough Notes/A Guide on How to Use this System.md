### Obsidian Vault:
- Use `Ctrl + O` then `Shift + Enter` to create a new note.
- Press `Alt + T` on the newly created page to apply the quartz template.
### Terminal
- Use `npx quartz build --serve` to run localhost live view.
- Use `npx quartz sync` to sync on GitHub.
